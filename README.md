# Fish Detection Frontend

방어, 부시리, 잿방어 분류 AI 서비스 프론트엔드

## 🚀 Tech Stack

- React.js / Next.js
- TypeScript
- TailwindCSS
- Axios (API 통신)

## 📁 Project Structure

```
frontend/
├── public/
├── src/
│   ├── components/     # 재사용 컴포넌트
│   ├── pages/          # 페이지
│   ├── services/       # API 호출
│   ├── utils/          # 유틸리티
│   └── App.jsx
├── package.json
└── README.md
```

## 🛠️ Setup

```bash
npm install
npm run dev
```

## 🔗 API Endpoint

Backend: `http://localhost:8000`
